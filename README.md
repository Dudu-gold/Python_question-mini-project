# Python_question-mini-project

A simple Python project that demonstrates how to build and run a question-based program using **object-oriented programming** and **multiple files**.

The project separates concerns by:
- Defining a `Question` class in one file
- Importing and using that class in another file that contains the questions and game logic

This makes the code easier to read, maintain, and extend.
